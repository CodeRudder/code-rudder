## 项目目标(一句话)
本项目实现一个AI编程助手插件，通过Hook机制自动管理开发流程、维护项目记忆文件，实现任务全流程自动化连续处理。

## 项目功能描述(200字以内)
Code Rudder是一个Claude Code插件，通过拦截StopHook等关键节点，注入智能开发流程管控规则。插件可自动生成执行计划、动态决策下一步操作、维护多种本地记忆文件（PRD/PLAN/CONTEXT/BUGS等），并自动进行单元测试和集成测试，实现从需求分析到代码提交的全流程自动化处理，大幅提升AI编程效率和质量。

## 关键需求（200字以内）
1. Hook拦截机制：拦截StopHook、StartHook等关键操作点
2. 智能决策规则：基于业务目标和开发流程自动决策下一步行动
3. 本地记忆管理：自动维护PRD、PLAN、CONTEXT、BUGS等多种记忆文件
4. 测试自动化：自动执行单元测试、接口测试、集成测试
5. 问题自动修复：识别并自动修复开发中的错误和Bug
6. 验收标准管控：确保所有功能通过验收标准后才停止执行
